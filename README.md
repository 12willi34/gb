# simplegb

Simple emulator for the original GameBoy written in golang,  
compatible with most GameBoy games.  

<br><video src="doc/demo.mov" width=180/>

## run a game

```
git clone git@github.com:12willi34/simplegb.git
cd simplegb/src/sdl
go build -o simplegb
simplegb path/to/game.gb
```
